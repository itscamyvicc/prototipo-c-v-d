Importar / Exportar Dados

Gerencie seus dados de profissionais e documentos em massa.

Importar Dados

Faça upload dos dados dos profissionais e documentos usando um único CSV.

Tipo de arquivo: CSV separado por ponto e vírgula (;)

Campos obrigatórios do profissional:

nome cpf cargo setor email telefone dataAdmissao status

Campos obrigatórios do documento:

tipoDocumento numeroDocumento orgaoEmissor dataEmissao dataValidade

Exportar Dados

Gere um arquivo CSV com seus dados cadastrados.

Nota: Os arquivos CSV podem ser abertos no Excel para análise ou edição.

O arquivo CSV gerado contém profissionais e documentos vinculados, podendo ser usado para backup, auditoria ou reimportação no sistema.
